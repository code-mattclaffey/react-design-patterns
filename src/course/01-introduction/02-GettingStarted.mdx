# Getting Started

> Node version 18+ required.

## Installation

### SSH

```
git clone git@github.com:code-mattclaffey/react-design-patterns.git
```

### HTTP

```
git clone https://github.com/code-mattclaffey/react-design-patterns.git
```

### Next steps

```
npm i
npm run storybook
```

Once installed, you need to be on Node 18+ and then you just run an npm install and npm run storybook and away we go. Now let's look at how the lessons will be structured.
